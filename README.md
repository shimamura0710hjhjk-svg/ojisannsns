# 🧔 おじさんコンバーター

普通の文章をキモいおじさん構文に変換するWebアプリケーション

## 機能

### 🧔 おじさんモード

- 敬語をタメ口に自動変換
- 感情キーワードに応じた絵文字を自動付与
- ポジティブな感情：😘♡ キスマーク＆ハート
- ネガティブな感情：😭💢🙄 泣き&怒り&つかれた
- 短い文は自動で"盛る"処理
- 食べ物や場所に関連したグルメ絵文字を自動追加

## 使用方法

### 方法1: 自動セットアップスクリプトで実行（推奨）

```bash
./start.sh
```

このスクリプトが以下を自動で実行します：

- Pythonバージョンの確認
- 依存パッケージのインストール
- アプリケーションの起動

### 方法2: 手動で実行

```bash
# 依存パッケージをインストール
pip install -r requirements.txt

# アプリケーションを実行
python run.py
```

その後、ブラウザで `http://localhost:5000` にアクセスしてください。

### 方法3: Pythonスクリプトから直接実行

```bash
python3 run.py
```

## おじさんの特徴

- 敬語を使わないタメ口（ヨ、だな、かな、など）
- 感情表現が豊か（😘♡💛😭💢🙄）
- グルメな話題に強い（食べ物絵文字の自動挿入）
- 短い返答を自動で拡張
- 話題に応じた絵文字を末尾に追加

## 感情キーワード

### ポジティブ（おじさん）

楽しい😘、嬉しい😍、好き😘、恋😍、感謝😍、など

### ネガティブ（おじさん）

悲しい😭、辛い😭、寂しい😭、怒り💢、つかれた🙄など
